# Misaka v1.0.0-alpha.1

首个公开版本发布，Misaka 现已具备较完整的 Claude Code 桌面 GUI 使用体验。

**Release period:** 2026-02-26 to 2026-03-07

## Highlights

### Chat experience
- 支持多轮流式对话，回答可实时显示并持续更新
- 支持消息重新生成与回复复制，提升日常使用效率
- 支持文件路径插入、图片附件与更丰富的聊天上下文输入
- 优化消息列表、消息项和流式更新体验，让长回复更平滑稳定

### Session and workflow
- 建立会话管理基础能力，提升不同任务场景下的对话组织效率
- 优化会话列表、上下文菜单与聊天页面布局，日常操作更清晰顺手
- 增强文件树与聊天联动能力，更适合代码仓库协作场景

### Settings and extensibility
- 新增 Claude Code Router 配置管理与模型选择能力
- 增强调试日志、权限控制与多环境兼容性支持
- 建立技能管理基础，为后续扩展能力提供支持

### UI and localization
- 支持主题模式切换与强调色自定义
- 优化按钮、面板、表单弹窗和多个核心页面的视觉一致性
- 持续完善国际化与本地化内容，提升多语言使用体验

## Fixes
- 修复流式会话结束后的异步清理问题
- 统一图片预览覆盖层行为，提升不同入口下的查看一致性
- 修复部分权限卡片与界面布局细节问题

## Documentation and release readiness
- 完善 README、安装说明、使用示例和快速指南
- 补充中英文文档入口与产品特性说明
- 更新许可证信息并完善首个版本发布基础

## Notes
Misaka 是一个基于 Python + Flet 构建的 Claude Code 桌面客户端，聚焦于将命令行能力带入更直观、现代且适合日常使用的桌面图形界面。
